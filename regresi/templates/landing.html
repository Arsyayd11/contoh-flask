Apa itu Interval Kepercayaan pada Regresi?
Berbeda dengan klasifikasi yang memiliki confidence score, pada regresi kita menggunakan interval kepercayaan ±MAE.

Artinya: jika model memprediksi harga $200,000 dengan MAE ${{ "{:,}".format(mae_usd) }}, maka harga sesungguhnya diperkirakan berada di antara ${{ "{:,}".format(200000 - mae_usd) }} hingga ${{ "{:,}".format(200000 + mae_usd) }}.
Fitur yang Digunakan
RM
Rata-rata jumlah kamar per unit hunian. Semakin banyak kamar → harga cenderung lebih tinggi.
LSTAT
Persentase penduduk kelas bawah di area tersebut. Semakin tinggi % → harga cenderung turun.
PTRATIO
Rasio murid per guru di sekolah sekitar. Semakin tinggi rasio → kualitas pendidikan lebih rendah → harga cenderung turun.
CHAS
Apakah properti berbatasan dengan Sungai Charles? (1=Ya, 0=Tidak). Lokasi pinggir sungai biasanya lebih mahal.
RAD
Indeks aksesibilitas ke jalan raya radial (1=sangat rendah, 8=sangat tinggi). Lebih mudah dijangkau umumnya lebih bernilai.
Tentang Dataset
Dataset Boston Housing berisi informasi harga properti di area Boston, Massachusetts, USA. Dataset ini klasik dalam dunia machine learning dan sering digunakan untuk contoh regresi.

Model dilatih menggunakan Gradient Boosting Regressor — algoritma ensemble yang membangun pohon keputusan secara berurutan, di mana setiap pohon baru mencoba memperbaiki kesalahan pohon sebelumnya.
